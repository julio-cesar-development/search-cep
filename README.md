> Feito com javascript, gulp, sass  
> Usa uma API de busca de CEP

## Instruções

> Rode o comando:

```bash
npm install
```

> E após isto o seguinte comando:

```bash
npm run gulp
```

> E basta abrir o arquivo index.html diretamente via protocolo file:// em seu browser
